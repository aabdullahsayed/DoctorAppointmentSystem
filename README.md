
# Doctor Appointment System

Doctor Appointment Management System built using ASP.NET Core MVC that allows patients to book appointments.

## Technologies Used
* **Framework:** ASP.NET Core MVC

* **Database:** SQL Server

## Project Structure
* **Controllers:** Handles application routing
* **Data:** Contains the `ApplicationDbContext` and registered `DbSets` for database management.
* **Models:** Defines the core domain models and their relationships.
* **Views** Stores the user interface views.
* **wwwroot:** Houses the static assets (CSS, JS, images).

## Getting Started
1. Clone the repository 
2. Navigate to `appsettings.json` and configure your SQL Server connection string.
3. Ensure the launch settings in the `Properties` folder are configured to your environment.
4. Run the project to start the application and book appointments.
